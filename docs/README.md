# Code Academy Documentation Package

This package contains the planning and prompt documents for the Eyal's Coding Academy web app.

## Hierarchy

```text
docs/
  planning/
    PRD.md
    TECHNICAL_REQUIREMENTS.md
    TASK_BREAKDOWN.md
  prompts/
    README.md
    00_VERIFY_EXISTING_IMPLEMENTATION.md
    01_DATABASE_SCHEMA_AND_RLS.md
    02_YOUTUBE_PARSER_AND_METADATA.md
    03_HOME_PAGE_AND_CATALOG.md
    04_COURSE_PAGE_AND_PROGRESS.md
    05_ADMIN_COURSE_MANAGEMENT.md
    06_AI_TUTOR_CONTEXT_AND_PERSISTENCE.md
    07_DASHBOARDS.md
    08_LOCALIZATION_SEO_ACCESSIBILITY_POLISH.md
    09_TESTING_AND_QA.md
    10_DEPLOYMENT_AND_FINAL_REVIEW.md
```

## Usage

Copy the `docs` directory into the root of the project repository. Start with `docs/prompts/00_VERIFY_EXISTING_IMPLEMENTATION.md`, then proceed in order.

Task 0 is intentionally verification-first. The project already has a working template foundation, so the implementation plan avoids unnecessary rewrites.
